---
title: 'Reset password'
description: 'Set a new password using the reset token received via email after requesting a password reset.'
---

Set a new password for the authenticated user. You must pass the reset token from the password reset email as a Bearer token in the `Authorization` header.

## Request

**`POST /api/v1/auth/reset-password`**

Requires the reset token from the password reset email as a Bearer token.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token from the password reset email link. Format: `Bearer <resetToken>`. The middleware uses this to authenticate the request.
</ParamField>

### Body

<ParamField body="token" type="string" required>
  The reset token from the password reset email link. This is validated by the request body schema in addition to the `Authorization` header.
</ParamField>

<ParamField body="password" type="string" required>
  The new password for the account. Must meet all of the following requirements:

  - At least 8 characters
  - At least one uppercase letter (A–Z)
  - At least one lowercase letter (a–z)
  - At least one number (0–9)
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indicates whether the request succeeded. Always `true` on success.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="message" type="string">
      Confirmation message. Value: `"Password reset successfully"`.
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://your-api.com/api/v1/auth/reset-password \
  --header 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...' \
  --header 'Content-Type: application/json' \
  --data '{
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "password": "NewSecureP4ss"
  }'
```

```typescript TypeScript
// Extract the reset token from the URL query parameter sent via email
const urlParams = new URLSearchParams(window.location.search);
const resetToken = urlParams.get('token');

const response = await fetch('https://your-api.com/api/v1/auth/reset-password', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${resetToken}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    token: resetToken,    // required by request body schema
    password: 'NewSecureP4ss',
  }),
});

const data = await response.json();
// { success: true, data: { message: 'Password reset successfully' } }
```

</CodeGroup>

### Success response

```json 200
{
  "success": true,
  "data": {
    "message": "Password reset successfully"
  }
}
```

### Error responses

```json 400
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Password must be at least 8 characters"
  }
}
```

```json 401
{
  "success": false,
  "error": {
    "code": "UNAUTHORIZED",
    "message": "Invalid or expired token"
  }
}
```
